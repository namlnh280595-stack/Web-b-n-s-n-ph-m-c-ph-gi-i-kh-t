# lnhnam
Dự án trang web bán sản phẩm cà phê giải khát
Đây là dự án đầu tay của mình trong quá trình học lập trình web với ASP.NET Core. 
Dự án này tập trung vào việc quản lý danh sách sản phẩm và hiển thị thực đơn đồ uống.
Giới thiệu
Dự án này được xây dựng để thực hành các kiến thức cơ bản về mô hình MVC (Model-View-Controller) và cách quản lý dữ liệu trong hệ sinh thái .NET.
Các chức năng chính
Quản lý sản phẩm: Hiển thị danh sách đồ uống với đầy đủ thông tin.
Giao diện dược thiết kế lấy cảm hứng từ không gian quán cafe ấm cúng, hiện đại.
Công nghệ sử dụng
Backend: C#, .NET Framework, Razor Syntax.
Database: Entity Framework (EF).
Frontend: HTML, CSS, JavaScript.
Quản lý mã nguồn: Git & GitHub.
Cách chạy dự án trên máy cục bộ
Tải mã nguồn
git clone https://github.com/namlnh280595-stack/Web.git
Chạy ứng dụng: Nhấn nút Start (phím tắt F5) để chạy trên IIS Express.
